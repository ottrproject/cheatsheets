Responsibilities

A session chair at a scientific conferences:

  • introduces topics and speakers
  • facilitates questions
  • manages time

Tasks Ahead of Time

  • meet with session speakers
    • check name pronunciation
    • check affiliation
    • remind speakers about declaring conflicts
    • explain timing
    • technical checks
